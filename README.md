# Unnamed O2 Clone "UO2C" - Resources
A repository to store resources for the Unnamed O2 Clone "UO2C" project.

## How to use it
1. Clone the [UO2C](https://github.com/estrol/o2game) repository recursively.
2. Compile the project.
